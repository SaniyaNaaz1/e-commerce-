# E-Commerce Analytics using Apache Spark

This project analyzes e-commerce sales data using Apache Spark.

## Features
- Data cleaning and validation
- Revenue calculation using Spark SQL
- Category & product analytics
- Visualization using Python (Matplotlib)
- Integrated PostgreSQL as analytics storage layer
- Validated data access using Python (psycopg2)


## Tech Stack
- Apache Spark
- Python
- PostgreSQL
- VS Code

## Output
- CSV analytics outputs
- Revenue charts for business insights
